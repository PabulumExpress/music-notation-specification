# Clef

* G clef (Treble)
G-clef符号围绕g线
* F clef (Bass)
F-clef符号围绕f线，两点位于f线上下
* C clefs (alto, tenor, baritone, soprano, mezzo-soprano)
中心点位于c线

## 改变clef
* Measure间改变
    * 放在将要改变的Measure的前一个measure的后bar line的前面
    * 多个Rest Measures的情况忽略非末尾的measure
* Measure中改变
    * 必须放在Rest之后
    * 在beam/slur/tie中不能改变
